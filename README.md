# Página de Star Wars feito por um fã.

---

## Sobre o projeto:

Um projeto acadêmico pensado para desenvolver minhas técnicas aprendidas em aula durante o primeiro semestre de 2026 na SPTECH.


### No website é possível:


Conhecer um pouco sobre o universo de Star Wars;

Fazer um Cadastro e Login;

Responder um Quiz para testar seus conhecimentos;

Entender a ordem cronológica do universo com uma Timeline;

Visualizar uma Dashboard dinâmica para acompanhar a classificação geral (de todos usuários) e a sua maior pontuação no Quiz.

---

## Tecnologias usadas:

Node.js;

JavaScript;

Canva;

MySQL Workbench
